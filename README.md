# Interactive Face Particle Visualization System

A high-performance, real-time interactive installation recreation of the digital shadow effect.
Built entirely in Python using OpenCV, MediaPipe, ModernGL, and GLSL.

## Milestone 1
- Project Architecture & Folder Structure
- Dependency Selection
- Environment Setup
- Initial Window (ModernGL + GLFW)
- Asynchronous Webcam Pipeline (OpenCV)

## Setup
1. Create a virtual environment:
   `python -m venv venv`
2. Activate the virtual environment:
   `venv\Scripts\activate` (Windows)
3. Install dependencies:
   `pip install -r requirements.txt`
4. Run the app:
   `python main.py`

Press `ESC` to exit the application.
